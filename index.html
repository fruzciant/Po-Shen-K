¿Qué es el método de Po-Shen Loh?

El método de Po-Shen Loh es un enfoque intuitivo para resolver ecuaciones cuadráticas. En lugar de memorizar fórmulas, este método se basa en las fórmulas de Vieta y la geometría de las raíces. Para una ecuación x² + bx + c = 0, las raíces se expresan como x = -b/2 ± u, donde u se calcula de forma natural a partir de la relación entre las raíces.
